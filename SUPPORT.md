# Support

Use [GitHub Issues](https://github.com/Arnie016/discord-agent-army/issues) for reproducible bugs, feature requests, and setup questions.

Include the plugin version, operating system, Node.js version, command used, and redacted error output. Never include Discord bot tokens, OpenAI API keys, private messages, or member data.

Security vulnerabilities should be reported privately as described in [SECURITY.md](SECURITY.md).
