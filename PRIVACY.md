# Privacy Policy

Effective: July 15, 2026

Discord Agent Army is a local, skills-only Codex plugin. The plugin itself has no hosted service, analytics, advertising, telemetry, or developer-operated database.

## Data handled

Generated Discord bots may process Discord user and server identifiers, messages sent to configured support channels, verification reasons, selected interests, and server configuration. This processing occurs in the operator's Discord application and infrastructure.

If an operator enables optional OpenAI-assisted FAQ responses, the support question and matched FAQ context may be sent to the OpenAI API. The template requests `store: false`; OpenAI's applicable data policies still govern that API request.

## Storage and sharing

The default template does not create a persistent database. Discord retains messages according to the server's settings and Discord's policies. The plugin developer does not receive or sell community data.

## Operator responsibilities

Each server operator is the controller of their deployment. Operators must provide appropriate notices, configure retention and permissions, respond to member privacy requests, and comply with Discord's terms and applicable law.

Questions can be filed through the project's public support channel: https://github.com/Arnie016/discord-agent-army/issues
